## Part 5: The Personalization Quiz (Killer Feature)

### Why This Is Our Moat

Based on our scientific research, **individual response to noise varies dramatically** based on:
1. **ADHD/dopamine levels** - Need MORE noise for optimal performance
2. **Age** - Older adults benefit most from pink noise for SWS
3. **Noise sensitivity** - High neuroticism = may be harmed by noise
4. **Environment** - Urban dwellers benefit more from masking
5. **Use case** - Sleep vs focus require different approaches

**No competitor offers this.** Everyone treats users as identical.

### Quiz Flow Design

```
Screen 1: "What's your main goal?"
[ ] Better sleep
[ ] Improved focus/productivity
[ ] Both

Screen 2: "How would you describe your typical environment?"
[ ] Quiet (rural/suburban)
[ ] Moderate noise (some traffic, neighbors)
[ ] Noisy (urban, shared living, snoring partner)

Screen 3: "Do you experience any of these?" (multi-select)
[ ] Difficulty concentrating
[ ] Racing thoughts at bedtime
[ ] ADHD diagnosis or traits
[ ] Sensitivity to sounds
[ ] None of the above

Screen 4: "What's your age range?"
[ ] Under 30
[ ] 30-50
[ ] 50-65
[ ] 65+

Screen 5: "Have you tried noise for sleep/focus before?"
[ ] Yes, it helped
[ ] Yes, it didn't work for me
[ ] No, I'm new to this
```

### Personalized Output

Based on responses, provide:
1. **Recommended noise type** (e.g., "Pink noise is likely your best match")
2. **Starting volume** (adjusted for sensitivity + age)
3. **Mode suggestion** (Sleep vs Focus)
4. **Scientific explanation** (why this matches them)
5. **Experimentation tips** (try other types if this doesn't work)

## Part 7: Technical Differentiators

### Real-Time Noise Generation (No Loops)

**Why this matters:**
- User complaint: "I can hear when it restarts"
- Looping is the #1 technical complaint about noise apps
- Real-time generation = truly infinite, never repeats

**Implementation (from our earlier research):**
- flutter_soloud for real-time PCM streaming
- Pink noise: IIR filter with coefficients B=[0.049922035, -0.095993537, 0.050612699, -0.004408786]
- Brown noise: Leaky integrator with coefficient ~0.02
- Sample rate: 44100 Hz, buffer: ~50ms chunks

### Volume Safety System

**Unique feature - no competitor does this properly:**
- Real-time dB estimation
- Visual indicator (green/yellow/red)
- Warning modal if >70 dB sustained
- Baby mode: Hard cap at 50 dB
- Distance recommendations for infant use
- Educational tooltip: "Volumes above 70dB can affect hearing over time"
